OBPieChartConfig Options

Name Type Default Description
width number | function 1024 Width of the SVG Graph
margin object 20px, 20px, 20px, 20px Margin of the graph for each side. Object structure is : margin: {top : 20 ,right : 20,bottom : 20, left : 20}
outerRadius number 120

Outer Radius of Primary Ring

Note: This is also used to calculate the height of the Graph. height = (outerRadius * 2) + margin.top + margin.bottom
innerRadius number 80 Inner Radius of Primary Ring
collapseOuterRadius number 81 Outer Radius of Primary Ring when collapsed (During Hover over "Other")
collapseInnerRadius number 60 Inner Radius of Primary Ring when collapsed
sunburstOuterRadius number 120 Outer Radius of Secondary ("Other") Ring
sunburstOuterRadius number 80 Inner Radius of Secondary ("Other") Ring
colors array ['#FFDFCC','#FFA066', '#FF6000' ,'#BF4800','#200C00'] List of Hex color strings used for the primary ring. Example ['#FFDFCC','#FFA066', '#FF6000' ,'#BF4800','#200C00']
collapseColors array ['#D9D1CC', '#B2A399','#8C7466','#664633','#200C00'] List of Hex color strings used for the primary ring while the ring is collapsed. Example ['#D9D1CC', '#B2A399','#8C7466','#664633','#200C00']
outerColorRange array ['#CEE7ED', '#08C1EA', '#200C00'] List of 3 colors used for the outer ring's color interpolator. color = ['#CEE7ED','#08C1EA','#200C00']

Given 3 data points:

Each slice will get a 1-1 assignment for color.

Given 5 data points:

Slices 1, 3, 5 will get be color[0],color[2],color[4]

Slice 2 will get an interpolated color between color[0],color[2]
keyAccessor string, function "label" Accessor function used to find a pie slice's unique key. If value is a string, the each slice's data object will be accessed directly by that string.
With Slice data_object = {value : 4, key: '1542', label : 'CNN.com'} and keyAccessor="key", the unique key would be data_object["key"]
If the value is a function, it should return the unique key when passed the slice's data object. Example: function(d){return d.some_object.nested_field}
